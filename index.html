API for International Space Station

This API gets telemetry for the International Space Station. It has data points for the positon, velocity, and conditions on the ISS.
Link to API Documentation: https://wheretheiss.at
The options I used for this API request is using the GET method inside AJAX to get the JSON object at the URL. Then after getting the JSON object it returns it and parses the JSON object, formatting all data points on the page.
This API could be helpful if someone wanted to make a website that montiored satellite positions and track there telemetry. One other potential use for an API like this would be as a means of creating a data source for all public or private satellite positions to prevent collisions. For example, if SpaceX put a satellite in orbit it can share the satellites telemetry with the API so when NASA wants to put another satellite into orbit, it can easily check with the API's live data points to make sure a collision or interference error does not occur.
